---
description: comment vérifier et corriger les traductions (i18n)
---

Si l'utilisateur mentionne une vérification de traduction ou le dossier `scripts/i18n` :

1.  **Vérifier la cohérence** :
    - Lance `node scripts/i18n/verify.mjs`.
    - Si des erreurs sont trouvées, propose de les corriger en utilisant une "map" de traduction.
2.  **Trouver les clés manquantes dans le code** :
    - Lance `node scripts/i18n/extract.mjs`.
    - Si des clés roses (ex: `NAMINGNOTICE`) apparaissent dans l'UI ou si le script en trouve, ajoute-les à tous les fichiers `/messages/*.json`.
3.  **Apporter des corrections** :
    - Toujours mettre à jour les 12 langues simultanément pour garder la cohérence.
    - Utiliser les scripts pour valider tes changements avant de terminer.
